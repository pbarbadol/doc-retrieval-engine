# Doc-Retrieval-Engine

A Retrieval-Augmented Generation (RAG) system built to demonstrate core AI Engineering skills, including document processing, vector embedding, semantic search, and LLM integration.

Status: Work in Progress. This repository is actively being developed. Results, evaluations, and optimizations will be documented as they are completed.

## Tech stack
* Language: Python
* Framework: [LangChain]
* Vector database: [Chroma] (by the moment)
* LLM: Local, mistral with ollama
